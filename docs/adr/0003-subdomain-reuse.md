# 0003 Subdomain Reuse

Date: 2026-02-04

## Context

Нужно минимизировать время выдачи поддомена и использовать уже выпущенные сертификаты, если поддомен свободен.

## Decision

- Вести файл-реестр известных поддоменов (`SUBDOMAIN_STORE_PATH`).
- При регистрации, если поддомен не запрошен явно, переиспользовать свободные из списка.
- При отсутствии свободных генерировать новый и добавлять в реестр.

## Consequences

- Реестр растет со временем, но позволяет быстро переиспользовать уже выпущенные домены/сертификаты.
