# 0001 Initial Scope

Date: 2026-02-04

## Context

Нужно создать минимальный аналог ngrok на Go с двумя бинарниками (сервер и клиент), с фокусом на 12-factor и минимальные зависимости.

## Decision

- Реализовать только HTTP/HTTPS туннелирование.
- Авторизацию пока не добавлять.
- Использовать только stdlib.
- Конфигурацию полностью через переменные окружения.

## Consequences

- Нет поддержки TCP/UDP raw и прочих протоколов.
- Без авторизации сервис предназначен для внутреннего использования.
