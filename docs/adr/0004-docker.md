# 0004 Docker Support

Date: 2026-02-04

## Context

Нужно обеспечить возможность запуска сервера и клиента в Docker контейнерах с предсказуемой конфигурацией.

## Decision

- Добавить отдельные Dockerfile для сервера и клиента.
- Добавить инструкцию по запуску (build/run) в README.

## Consequences

- Появляется дополнительный способ запуска (Docker) без изменения кода.
- Для запуска в контейнере требуется передавать все конфигурации через переменные окружения и проброс портов.
